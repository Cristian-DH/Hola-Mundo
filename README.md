# Hola-Mundo
Investigar como escribir un hola mundo en tres lenguajes diferentes 

•	En python: 

print("¡Hola, mundo!")

•	En Java:

public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola Mundo");
    }
}


•	En Javascript:

console.log("Hola Mundo");


•	En C++:

#include <iostream>

int main() {
    std::cout << "Hola Mundo" << std::endl;
    return 0;
}


•	¿Qué hace el programa Hello World?

El programa hello world sirve para generar texto en pantalla o consola.


•	¿Qué diferencias visuales notan entre JavaScript, Python y Java?

Tienen diferentes sintaxis. Python y javascript se nota a simple vista que usa menos símbolos y abreviaturas, mientras que java usas más.

•	¿Qué creen que significa ‘sintaxis’ después de ver estos ejemplos?

Creo que es el conjunto de reglas dentro de un lenguaje especifico.
